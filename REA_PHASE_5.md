## 5A: Monthly Content Calendar System

### The Goal

SEO authority for a single-agent site is built through consistent, quality content over 12+ months. Two blog posts and one market report per month compounds into a powerful organic traffic moat. This is non-negotiable for long-term SEO success.

### Monthly Cadence (Every Month)

**Week 1:**
- Market Report: Publish monthly Orange County market update
  - Stats: median price, DOM, inventory, YoY change, list-to-sale ratio
  - 500–800 words Jin's analysis
  - Admin → Market Reports → Create
  - Share to: Google Business Profile post, social media

**Week 2:**
- Blog Post #1: SEO-targeted educational article
  - See content themes below for topic ideas
  - 800–1,200 words minimum
  - Admin → Knowledge Center → Create
  - Share to: GBP, Instagram, Facebook

**Week 4:**
- Blog Post #2: Local or personal-voice article
  - Neighborhood spotlight, seasonal tips, transaction story (anonymized)
  - 600–1,000 words
  - Admin → Knowledge Center → Create

**Every Quarter (Month 3, 6, 9, 12):**
- New Case Study: document a recent transaction
  - Obtain client consent
  - Admin → Case Studies → Create
  - Include in next market report as "Featured Transaction"

**Every Quarter:**
- Testimonial sweep: collect new reviews from recent clients
  - Request on: Google, Zillow, direct
  - Add all to Admin → Testimonials

### Content Workflow — Cursor Prompt Template

For each blog post, use this Cursor prompt:

```
Write a blog post for Jin Pang's real estate site (jinpanghomes.com).

Post title: [TITLE]
Category: [CATEGORY]
Target keyword: [PRIMARY KEYWORD]
Secondary keywords: [KEYWORD 2], [KEYWORD 3]
Target length: 900–1,200 words

Tone: First person, expert but approachable. Jin Pang's voice:
- Local authority (I know this market from the inside out)
- Honest and direct (no spin, real numbers when possible)
- Personal (share relevant personal knowledge or experience)
- Practical (readers can do something with this information)

Structure:
- H1: [post title — include primary keyword naturally]
- Introduction: 2–3 sentences, hook the reader, state what they'll learn
- H2 sections: 4–6 subheadings, each with 150–200 words

- Include internal links to relevant service pages:

- End with a clear CTA paragraph:
  "If you have questions about [topic], I'm happy to talk through your
  specific situation. Call me at (845) 555-0142 or [book a consultation →]."

After writing: provide SEO meta title (60 chars) and meta description (155 chars).
```

## 5C: Conversion Rate Optimization (CRO)

### CRO Cadence: Quarterly Sprints

Every 3 months, run a 2-week CRO sprint targeting one conversion funnel.
Use GA4 data + Vercel Analytics to identify biggest drop-off points first.

### Metrics to Track (Set up in GA4)

Configure these GA4 events on launch:

```javascript
// Key conversion events to track via gtag()
gtag('event', 'contact_form_submit', { form_type: 'contact', locale: 'en' })
gtag('event', 'valuation_form_submit', { form_type: 'valuation' })
gtag('event', 'showing_request_submit', { property_slug: slug })
gtag('event', 'gated_download_submit', { guide_slug: slug })
gtag('event', 'investor_inquiry_submit', { inquiry_type: type })
gtag('event', 'cta_click', { cta_location: 'hero', cta_label: 'Book a Consultation' })
gtag('event', 'phone_click', { location: 'topbar' })
```

### CRO Sprint 1 (Month 3) — Home Page Conversion

**Question:** What % of home page visitors click a CTA?

GA4 funnel: home page view → intent path click OR CTA click → contact/valuation form start → form submit

Common improvements to test:
- Hero CTA button color: test champagne gold vs primary teal
- Intent paths order: test Buy/Sell/Invest vs Sell/Buy/Invest
- Personal intro photo: test different agent photo (lifestyle vs formal)
- Stats bar values: test animated count-up vs static display

Implementation: Use Next.js A/B testing approach:
- Create two variants of the section in content JSON (variant A and B)
- Use a cookie to consistently serve same variant to same user
- Track events per variant in GA4

### CRO Sprint 2 (Month 6) — Contact Form Funnel

**Question:** Why are visitors landing on /contact but not submitting?

Audit:
- Form length: are all fields necessary? Can any be optional?
- Form placement: is it visible above fold on mobile?
- Friction points: does the "purpose" dropdown feel interrogative?
- Trust signals near form: add testimonial quote beside form
- Response promise: is "2 hours" specific enough?

Tests to run:
- Test: shorter form (name, phone, email, message only — no purpose dropdown)
- Test: add a 5-star review excerpt next to form submit button
- Test: change submit label from "Send Message" to "Get a Free Consultation"

### CRO Sprint 3 (Month 9) — Gated Download Conversion

**Question:** What % of guide card viewers submit the gate form?

Audit:
- Is the guide value proposition clear? (What will they learn?)
- Is the cover image compelling?
- Is the gate form too long? (reduce to email-only and test)
- Add urgency: "Download the guide 120 families used before moving to Orange County"

### CRO Sprint 4 (Month 12) — ZH Locale Conversion

**Question:** Are Chinese-speaking visitors converting at the same rate as EN visitors?

Metrics: compare EN vs ZH for: contact form submit rate, phone click rate, time-on-page

If ZH converts lower:
- Check: are all ZH form labels accurate? (get native speaker review)
- Check: is WeChat contact method prominent enough on ZH pages?
- Check: does the ZH about page establish enough trust?
- Consider: add dedicated ZH landing page with WeChat QR as primary CTA

## 5D: Lead Nurture + Email System

### Current State at Launch

All lead capture points funnel into Supabase tables. Jin Pang is notified by email. No automated follow-up sequences exist at launch. Phase 5 adds structured nurture.

### Month 3 — Basic Email Sequences via Resend

**Sequence 1: Gated Download Follow-Up (3 emails)**

Trigger: gated_download submitted for any guide

Email 1 (Day 0 — immediate): Guide delivery + personal intro
  Subject: "Your [Guide Name] from Jin Pang — and a quick note"
  Body: Guide PDF link + 3-sentence personal note from Jin + phone number

Email 2 (Day 3): One useful tip related to guide category
  Subject: "One thing most [buyers/sellers/investors] overlook"
  Body: 200-word educational tip + soft CTA "Questions? I'm here."

Email 3 (Day 7): Offer to talk
  Subject: "Ready to talk through your situation?"
  Body: Brief check-in + direct CTA to book consultation or call

Implementation:
  Use Resend scheduled emails + a simple send-queue in Supabase:
  Table: email_queue (id, to_email, template_id, send_at, sent, created_at)
  Cron: Vercel Cron → /api/admin/email-queue-process (runs every 5 minutes)
  On gate form submit: insert 3 rows into email_queue with send_at timestamps

**Sequence 2: Home Valuation Follow-Up (2 emails)**

Trigger: valuation_request submitted

Email 1 (Day 0 — immediate): Acknowledge + set expectations
  Subject: "Got your valuation request — here's what happens next"
  Body: What Jin will do, 24-hour delivery promise, phone number

Email 2 (Day 1 — if no manual follow-up logged): Gentle nudge
  Subject: "Your Orange County home value report is ready"
  Body: "I've completed your report — let's schedule 15 minutes to walk through the numbers."
  Note: only send if CRM field "valuation_contacted" is not set

**Sequence 3: Monthly Newsletter (ongoing)**

All newsletter_subscribers + opted-in lead contacts receive:
Monthly: Orange County Market Snapshot (brief — 5 bullets + "Read full report" link)
Quarterly: "What's Happening in Orange County" digest

Build via Resend broadcast API:
  /api/admin/send-newsletter → batch send to newsletter_subscribers where active = true
  Admin trigger: Admin → Newsletter → Send Monthly Update button

### Month 6 — CRM Integration (if needed)

If lead volume exceeds what Jin can manage in Supabase admin UI:
- Connect to a real CRM: Follow Up Boss, Chime, or LionDesk (common in RE)
- Configure crmWebhookUrl in site.json integrations
- All new leads begin syncing to CRM automatically via existing CRM sink

## 5E: GSC Monitoring + SEO Iteration

### Monthly GSC Review (30 minutes)

On the 1st of every month, review Google Search Console:

**1. Coverage Report:**
  - Any new "Errors" or "Valid with warnings"?
  - "Excluded" pages: should any of these be indexed?
  - If new errors appear: fix in code + resubmit via URL Inspection

**2. Performance Report (last 28 days vs prior 28 days):**
  - Total clicks: up or down?
  - Top queries: what are people finding us for?
  - New queries appearing: are they relevant? Worth writing content about?
  - CTR by page: any page with high impressions but low CTR?
    If yes: rewrite meta title/description for that page to be more compelling

**3. Rich Results:**
  - Check "Enhancements" tab: FAQ, Article, RealEstateListing status
  - Any errors in rich results: fix schema.org markup

**4. Core Web Vitals:**
  - "Poor" URLs: fix immediately
  - "Needs improvement" URLs: schedule optimization

### SEO Response Matrix

| GSC Signal | Action |
|---|---|
| Query with 50+ impressions, 0 clicks | Rewrite meta title to be more compelling |
| Query ranking position 11–20 | Strengthen existing page for that query: add content, improve internal links |
| Query not covered by any page | Write a new blog post targeting it |
| Page dropping in rankings | Check: has content changed? Did a competitor publish? Update and improve |
| Chinese queries appearing | Strengthen corresponding ZH page |
| New neighborhood queries | Create or expand that neighborhood guide |

### Quarterly Content Audit (1 hour)

Every 3 months, audit all published blog posts:
- Posts older than 6 months: are facts still accurate? Update if not.
- Posts with declining traffic: refresh with new data or expanded content
- Posts with growing traffic: consider expanding into a content cluster
  (add related posts that interlink with the growing post)

### 6-Month Link Building Sprint

External links pointing to jinpanghomes.com increase domain authority and rankings.

Month 6 link building activities:
- Submit to local business directories:
  Orange County Chamber of Commerce, Port Jervis area directories, NY DOL licensee search

  NAR member directory, OneKey MLS member directory
- Community sponsorships:
  Sponsor a local event → get listed on event website with a link

  Write a "moving to Orange County" piece for a NYC-area relocation blog

  Journalists covering real estate will often link back to the source
